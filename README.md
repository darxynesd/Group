# Портал Групи

Django веб-додаток для управління груповою діяльністю: форум, електронний щоденник, події, опитування та багато іншого.

## Структура проекту

Проект складається з наступних додатків:

- **home** - Головна сторінка з віджетами
- **accounts** - Система автентифікації та профілі користувачів
- **forum** - Форум для обговорень
- **gradebook** - Електронний щоденник з оцінками
- **events** - Події та календар подій
- **surveys** - Система опитувань
- **voting** - Система голосувань
- **announcements** - Оголошення
- **materials** - Корисні матеріали та посилання
- **portfolio** - Портфоліо студентів
- **gallery** - Галерея фото та відео

## URL Маршрути

- `/` - Головна сторінка

- `/gradebook/` - Електронний щоденник
- `/events/` - Події та календар
- `/surveys/` - Опитування
- `/voting/` - Голосування
- `/announcements/` - Оголошення

## Встановлення та запуск

1. Встановіть залежності:
```bash
pip install -r requirements.txt
```

2. Виконайте міграції бази даних:
```bash
python manage.py migrate
```

3. Створіть суперкористувача (адміністратора):
```bash
python manage.py createsuperuser
```

4. Запустіть сервер розробки:
```bash
python manage.py runserver
```

5. Відкрийте браузер та перейдіть на http://127.0.0.1:8000/

## Технології

- Django 5.2.8
- SQLite (за замовчуванням)
- HTML/CSS (Bootstrap опціонально)

## Ролі користувачів

- **Користувачі** - Базовий доступ до перегляду та створення контенту
